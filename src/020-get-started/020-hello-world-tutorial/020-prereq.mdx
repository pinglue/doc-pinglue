title: Environment setup
---

The first step would be to setup a Pinglue monorepo as explain in [this guide](:monorepo-setup). 

Once this is done, create a new project folder within the *packages* folder and call it *@hello-world*. 

In the following sections, within this folder we are going to create a number of Pinglue plugins and then using the Pinglue tools we bootstrap them into an app.
